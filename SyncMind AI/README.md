SyncMind AI | Full-Stack AI Productivity Platform
==============================================

**Tech Stack**: React, FastAPI, PostgreSQL, OpenAI API, Docker

SyncMind AI is a full-stack AI productivity platform that converts meeting transcripts and notes into structured action items, deadlines, owner assignments, and follow-up summaries for student teams and club workflows.

## Platform Capabilities

- **Transcript & Notes Ingestion**: Backend APIs and database pipelines built with FastAPI and PostgreSQL to support transcript ingestion, task generation, workspace management, and persistent user-specific project tracking.
- **LLM-Powered Extraction & Summarization**: LLM-based extraction and summarization features integrated via the OpenAI API, with improved action-item generation quality through prompt iteration, output validation, and user feedback testing.
- **Team Workflow Dashboard**: React dashboard for reviewing summaries, editing assigned tasks, tracking deadlines, and managing team workflows across projects in a single interface.

## Product & UX

- **Product Definition**: Clear product requirements, MVP scope, and success metrics such as task completion rate, weekly active users, and summary usefulness, aligning technical development with user workflow needs.
- **User-Centered Iteration**: UX and feature prioritization driven by user interviews and testing with student organizations, translating user pain points into roadmap decisions and product improvements.
